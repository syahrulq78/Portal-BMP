⚙️ Setup Firebase

Tempel isi objek firebaseConfig dari Firebase Console. Konfigurasi disimpan di perangkat ini (localStorage) — tidak perlu edit kode.

Cara mendapatkan firebaseConfig:
1. Buka console.firebase.google.com
2. Pilih proyek → ikon ⚙️ Project Settings
3. Tab Your apps → klik ikon Web </>
4. Salin seluruh isi objek firebaseConfig = { … }
5. Pastikan Realtime Database sudah diaktifkan di proyek
