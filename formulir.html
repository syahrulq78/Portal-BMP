Referensi istilah yang digunakan dalam panduan ini dan dokumen SPMI BMP secara umum.

Singkatan / IstilahKeterangan
ABCD (pola standar)Format penulisan standar: A=Subjek, B=Kata Kerja/Kewajiban, C=Objek, D=Kondisi/Ukuran
Audit Mutu Internal (AMI)Proses pemeriksaan internal yang dilakukan PPM untuk memverifikasi pelaksanaan standar
BMPBagian Media dan Pers — unit kerja yang mengelola komunikasi dan publikasi IAI STIBA Makassar
Bukti EksekusiDokumen yang membuktikan bahwa rekomendasi perbaikan (RTL) bulan sebelumnya telah dilaksanakan
CTR (Click-Through Rate)Persentase orang yang mengklik sesuatu (thumbnail, tautan) setelah melihatnya
DOK-[kode]Kode penomoran dokumen dalam panduan ini. Contoh: DOK-3.1 = dokumen pertama di Bab 3
Dummy JurnalPrototipe atau versi awal Jurnal Akhbar yang digunakan sebagai bahan review redaksi sebelum dicetak
Editorial PlanKalender Konten Bulanan — dokumen perencanaan konten yang disusun di awal setiap bulan
Engagement RatePersentase audiens yang berinteraksi (like, komentar, simpan, bagikan) terhadap jangkauan konten
GA / GA4Google Analytics 4 — platform analitik gratis dari Google untuk memantau trafik website
HDD EksternalHard Disk Drive eksternal — perangkat penyimpanan fisik untuk backup data
IAI STIBAInstitut Agama Islam — Sekolah Tinggi Ilmu Islam dan Bahasa Arab Makassar
ImpressionsTotal tayangan konten; satu akun bisa dihitung lebih dari sekali
Indikator SMARTKriteria indikator kinerja: Specific, Measurable, Achievable, Relevant, Time-bound
Kabag. BMPKepala Bagian Media dan Pers
Kalender KolaborasiDokumen rencana kerja bersama BMP dan unit-unit lain dalam satu bulan
KPIKey Performance Indicator — indikator kunci kinerja yang diukur secara berkala
Log ProduksiCatatan operasional proses syuting podcast/video — dari setup hingga upload
LogbookDokumen rekam jejak harian untuk mencatat konten yang diunggah dan kegiatan dokumentasi
Lower-thirdTeks nama dan jabatan yang muncul di bagian bawah layar video saat narasumber berbicara
Matriks KompetensiTabel yang memetakan kemampuan setiap anggota tim BMP per bidang keahlian
Monthly InsightLaporan Kinerja Bulanan Media — dokumen analitik + rekomendasi yang diserahkan ke pimpinan
No. Arsip (ARF-)Nomor identifikasi unik untuk setiap aset foto/video dalam katalog arsip digital BMP
Notula RapatCatatan resmi hasil rapat yang ditandatangani pimpinan rapat dan notulis
NR-BMP/Nomor identifikasi Notula Rapat: NR-BMP/[NoUrut]/[BulanRomawi]/[Tahun]
PPEPPPenetapan, Pelaksanaan, Evaluasi, Pengendalian, Peningkatan — siklus penjaminan mutu SPMI
RTLRencana Tindak Lanjut — dokumen rekomendasi perbaikan berbasis data evaluasi bulanan
SPMISistem Penjaminan Mutu Internal — sistem pengelolaan mutu perguruan tinggi secara internal
ST-BMP/Nomor identifikasi Surat Tugas: ST-BMP/[NoUrut]/[BulanRomawi]/[Tahun]

⚙️ Pengaturan Firebase (Sinkronisasi Online)

Hubungkan ke Firebase Realtime Database agar data tersinkron secara real-time ke semua admin BMP. Gratis untuk kebutuhan ini.

📋 Cara Setup Firebase (satu kali, oleh admin utama)
1
Buka console.firebase.google.com → Add project → beri nama (misal: spmi-bmp-stiba) → Continue
2
Di menu kiri: Build → Realtime Database → Create Database. Pilih lokasi Singapore → Start in test mode → Enable
3
Setelah database terbuat, copy URL-nya. Bentuknya: https://spmi-bmp-stiba-default-rtdb.asia-southeast1.firebasedatabase.app
4
Paste URL tersebut di kolom Database URL di bawah. Isi nama admin Anda, lalu klik Simpan & Test
5
Bagikan file HTML ini + Database URL ke seluruh anggota tim. Setiap admin isi nama masing-masing di Pengaturan
Tanpa garis miring di akhir URL

📄 Simpan Formulir sebagai PDF

Ikuti langkah berikut untuk menyimpan formulir yang sedang ditampilkan sebagai file PDF:

1️⃣
Pastikan formulir sudah terisi dan halaman yang ingin disimpan sedang aktif ditampilkan
2️⃣
Tekan Ctrl + P (Windows) atau Cmd + P (Mac) untuk membuka dialog Print
3️⃣
Pada kolom Printer / Destination, pilih "Save as PDF"
4️⃣
Atur layout ke Portrait, Paper size: A4, centang Background graphics agar warna tetap tampil
5️⃣
Klik Save dan pilih lokasi penyimpanan
💡 Tips: Sidebar dan tombol tidak akan muncul di PDF karena sudah dikonfigurasi untuk tidak dicetak.
